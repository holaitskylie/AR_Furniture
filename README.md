Unity를 이용해 만든 AR 프로젝트입니다.  

AR Foundation을 이용하여 화면 내에서 현실 공간에 3D 가구 오브젝트를 배치할 수 있도록 만든 어플입니다.  

어플 내에서 지면을 인식하면 오브젝트가 나타나고 화면 상단의 슬라이더로 크기와 회전을 조정할 수 있습니다.
